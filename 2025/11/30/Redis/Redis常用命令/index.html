Redis常用命令

一、客户端连接

redis-cli -h host -p port -a password
输入命令：redis-cli -p 6379

二、数据导出导入

导出

执行备份命令：save
该命令将在 redis 备份目录中创建dump.rdb文件。
查看备份路径：config get dir
备份后的文件名：dump.rdb

导入

1、将dump.rdb拷贝到目标机，备份文件的目录下(如果开始不知道备份文件的目录在哪，可以使用Redis Desktop Manager软件在连接处打开控制台，输入save，看看生成的.rdb文件在哪，就知道备份文件的目录了)，覆盖式导入
2、重启redis服务
3、打开客户端刷新就能看到新的数据了

三、Redis string命令

# 增、改
set key value
mset k1 v1 k2 v2... //新增多个
# 删
del key
# 查
get key
mget k1、k2、k3... //查多个

四、Redis hash命令

# 增加一列、改
hset key 键 值
hmset key 键1 值1 键2 值2//增加多列
# 查
hget key 键
hgetall key //获取所有键对应的值
# 删
hdel key //删除整个hash
hdel key 键 //删除单个键

五、Redis list命令

lpush key value //左增    rpush key value //右增
lpop  key value //左删    rpop  key value //右删
rpush key v1 v2 v3... //插入多个
# 查所有
lrange key 0 -1

六、Redis set命令

# 增
sadd key value
# 查所有
smembers key
